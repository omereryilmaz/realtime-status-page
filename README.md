StatusPage is a realtime simple system monitoring application. It has been created using [Node.jsÂ®](https://nodejs.org/).

## Installation

Use the package manager [npm](https://www.npmjs.com) to install dependencies. In the project directory;

```bash
npm install
```

## Host List Configuration
The content of the `hosts.txt` file syntax is as follows:

```text
<System Name>=<URL or IP Address>
```


## Usage

To run the project:

```bash
npm start
```
